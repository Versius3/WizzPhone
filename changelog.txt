The WizzPhone 2.6 update and the first bug fixes have gone live, so the latest version is 2.8. It is important that there is a gear icon next to the ticket opening. Click on it and enter your data as if you were entering SD. The API part does not need to be checked yet, because it is not working yet. Then press the save button. If this step is done, only then open a ticket. Ticket opening is now completely automatic, everything happens in the background after you press the phone or email button. You can now open tickets for multiple users at the same time, enter the names separated by commas and then the system will open separate tickets for each user. It is important to fill in all the data for this. In the case of time, the system automatically enters minutes, so you just need to enter 4 there and then SD will take it as 4 minutes. Additionally, in the history section there is a separate button for setting macros and now there is a button where you can report bugs or write ideas for improving the app.

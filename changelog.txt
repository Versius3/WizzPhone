
After many months, WizzPhone has reached a major milestone. With version 2.0, the application has been updated with many new features, which you can read about in detail below.

v2.0 NEW FEATURES:

- The program will now update automatically. From now on, this application will always be updated, and a pop-up window will inform you if an update is available.
- The WizzPhone has moved under a completely new engine and has thus received a completely new design.
- Automatic ticket opening for phone and emails has been added. To do this, you will find 2 buttons at the bottom of the program. In the current situation, you need to press the Activites and then the Update Status button.     Within that, you need to select the time and the root cause. Everything else happens automatically. It is important that you save your login information the first time you log in so that you are logged in automatically.
- In the case of automatic ticket opening, the program resets the text, so you don't have to use the reset button after calls.
- The search bar has been added. When you search for an error, you will see errors found with that keyword. If there is only one result, the search bar will close.
- Compact mode has been added, a smaller mode that can be placed in the corner. All functions are available here too.
- Email template window has been added, which has pre-written email templates and you can also add your own custom templates. 
- You can now see your previous calls through the app.. This can be viewed by clicking on the gear icon. They can be copied with a click if needed.
- The Copy ID and Copy Text buttons have been added. ID copies only the ID, while Text copies the entire bottom text.
- When creating a ticket, it is now necessary to fill in the ID in each case, select the error, and also indicate whether it was a SELF or NSELF error.
- All log files and similar data files are accessible in encrypted form.

Thank you for your patience while the update arrived. From now on, you will receive information about all updates in this way.
